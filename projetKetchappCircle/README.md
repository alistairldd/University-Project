# This project replicates the game Circle from Ketchap.

This project was made alone for the course concurrent programming.

It was the first step toward a bigger project so that we learn how to better use threads.
A complete report will be available once the project is done.

# Playing the game 

Download the projet-pcii.jar file then double click on it.

if it doesn't work, open a terminal and run :

```bash
java -jar projet-pcii.jar
```

## **Prerequisite** : A java version **above Java 17** is necessary.  
> If the shell shows `UnsupportedClassVersionError`, you need to update your java version.
Run this command to check your java version
```shell
java -v
```
You need at least Java 17.0 to run it.


## Updating / Downloading Java (Windows)

```bash
winget install Oracle.JDK.21
```
